    public getMetaEvidence(contractAddress: string, metaEvidenceID: number, options: object): object: * source

    Get the MetaEvidence object for a metaEvidenceID. Hashes will be validated. By default MetaEvidence will be returned regardless of the validity of the hashes with an indicator on whether the hash was valid or not. To throw an error instead, use strict = true in options object. NOTE: If more than one MetaEvidence with the same metaEvidenceID is found it will return the 1st one.

    Return:

    object

    The metaEvidence object
