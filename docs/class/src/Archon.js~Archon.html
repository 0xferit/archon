    public constructor(ethereumProvider: string, ipfsGatewayURI: string) source

    Instantiates a new Archon instance that provides the public interface to Kleros contracts and library. All params are required. To use an individual portion of the API import a class and initialize it yourself.

    Params:

    NameTypeAttributeDescription
    ethereumProvider string

    The Web3.js Provider instance you would like the Kleros.js library to use for interacting with the Ethereum network.
    ipfsGatewayURI string

    The URI of a trusted IPFS gateway in order to fetch files from the IPFS network. Defaults to "https://gateway.ipfs.io"
